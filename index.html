Vault

Client-Side AES-256 Encryption

All operations run locally in your browser — nothing is transmitted

Cryptographic Details

Cipher: AES-256-GCM
Key Derivation: PBKDF2-SHA-256
Iterations: 600,000
Salt: 32 bytes random
IV: 12 bytes random
Auth Tag: 128-bit (built into GCM)
⚠
This tool runs entirely in your browser. No data is sent anywhere. For maximum security, disconnect from the internet before encrypting sensitive data, and clear this page when done. The passphrase is your only key — if lost, the data is unrecoverable.
